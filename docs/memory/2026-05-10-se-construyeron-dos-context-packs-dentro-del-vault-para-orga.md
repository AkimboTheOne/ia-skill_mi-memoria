## Memoria

- Tipo: convention
- Fuente: summary

## Resumen

Se construyeron dos context packs para orientar la organización documental:

1. raíz del proyecto desde `README.md`;
2. raíz documental desde `docs/`.

El `README.md` raíz quedó intacto. La documentación en `docs/` actúa como base de organización y referencia, no como runtime.

- Artefactos:
